Both forms are essentially the same, the only reason there are two is so you can see that the fields get appended to both of them. They make a POST request to httpbin, an awesome service that dumps all of the information about requests in a JSON format, so you can inspect them!

Form 1 - Has some fields predefined with classes, that will get filled with the appropriate information.






Form 2 - No fields predefined with classes, all fields will get created on load.
